# Online linear regression 

Code to solve linear regression models online using statmodels package in python. 


[![Open in Streamlit](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://online-linear-regression.streamlit.app/)

### How to run it on your own machine

1. Install the requirements

   ```
   $ pip install -r requirements.txt
   ```

2. Run the app

   ```
   $ streamlit run streamlit_app.py
   ```
